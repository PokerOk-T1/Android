## Регистрация и создание аккаунта в PokerOK через Android

Создание аккаунта в PokerOk через мобильное приложение на Андроид простое и быстрое. Для этого потребуется смартфон или планшет, подключение к интернету и установленный клиент. Процесс займет всего несколько минут, следуйте подробной инструкции ниже.

* Откройте актуальное зеркало PokerOk.
* В правом верхнем углу сайта найдите и нажмите кнопку «Регистрация».
* Выберите свою страну проживания из предложенного списка и подтвердите выбор.
* Заполните поля электронной почты и пароля, используя актуальные данные.
* Поставьте галочку, подтверждая, что вам исполнилось 18 лет и вы принимаете правила сервиса.
* Откройте электронное письмо, отправленное системой, и найдите в нем проверочный код.
* Внесите этот код в соответствующее поле на сайте, а затем нажмите кнопку «Подтвердить».
* Дождитесь уведомления о том, что ваша учетная запись успешно создана.

После регистрации игрок получает доступ ко всем функциям покерного клиента, включая участие в турнирах и кэш-играх.

## Вспомогательный скрипт pokerok.py

Скрипт `pokerok.py` на Python для базового анализа Android-приложений в формате APK. Скрипт предназначен для быстрого получения общей информации об APK-файле без его установки и без обращения к сети. Подходит для первичного анализа, реверса, forensics и проверки целостности APK. По умолчанию анализируется файл `pokerok.apk`, расположенный рядом со скриптом.

**Возможности:**

* Информация о файле:
  * размер
  * дата изменения
  * хэши (MD5 / SHA-1 / SHA-256)
* Проверка, что файл является APK (ZIP)
* Анализ структуры APK:
  * наличие `AndroidManifest.xml`, `classes.dex`, `resources.arsc`
  * определение multidex
  * поиск нативных библиотек (`.so`) и ABI
  * список самых больших файлов
* Анализ подписи (best-effort):
  * v1 (META-INF)
  * v2 / v3 (APK Signing Block)
  * проверка наличия v4 (`.idsig`)
* *(Опционально)* расширенный анализ через **androguard**:
  * package name
  * versionName / versionCode
  * minSdk / targetSdk
  * permissions
  * activities / services / receivers / providers

**Использование:**

```bash
python pokerok.py
```

Анализ конкретного файла:

```bash
python pokerok.py --apk app.apk
```

Вывод отчёта в JSON:

```bash
python pokerok.py --json
```

Сохранение JSON в файл:

```bash
python pokerok.py --apk app.apk --out report.json
```
